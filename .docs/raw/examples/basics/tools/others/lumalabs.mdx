## Code

```python cookbook/14_tools/lumalabs_tools.py
from agno.agent import Agent
from agno.tools.lumalabs import LumaLabsTools

agent = Agent(
    tools=[LumaLabsTools()],
        markdown=True,
)
agent.print_response("Generate a 3D model of a futuristic city")
```
