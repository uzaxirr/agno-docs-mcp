## Cloud Alternative

For easier setup without local installation, you can use [Ollama Cloud](/integrations/models/local/ollama/usage/cloud) with your API key:

```python
from agno.agent import Agent
from agno.models.ollama import Ollama

# No local setup required - just set OLLAMA_API_KEY
agent = Agent(model=Ollama(id="gpt-oss:120b-cloud", host="https://ollama.com"))
agent.print_response("Share a 2 sentence horror story", stream=True)
```
