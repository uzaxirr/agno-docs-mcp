LangDB is a AI Gateway for seamless access to 350+ LLMs. Secure, govern, and optimize AI Traffic across LLMs using OpenAI-Compatible APIs

## Parameters

| Parameter    | Type               | Default                        | Description                                                           |
| ------------ | ------------------ | ------------------------------ | --------------------------------------------------------------------- |
| `id`         | `str`              | `"gpt-4o-mini"`                | The id of the LangDB model to use                                    |
| `name`       | `str`              | `"LangDB"`                     | The name of the model                                                 |
| `provider`   | `str`              | `"LangDB"`                     | The provider of the model                                             |
| `api_key`    | `Optional[str]`    | `None`                         | The API key for LangDB (defaults to LANGDB_API_KEY env var)          |
| `base_url`   | `str`              | `"https://api.langdb.ai/v1"`  | The base URL for the LangDB API                                       |

| `retries`    | `int`              | `0`                            | Number of retries to attempt before raising a ModelProviderError      |
| `delay_between_retries` | `int`    | `1`                            | Delay between retries, in seconds                                     |
| `exponential_backoff` | `bool`     | `False`                        | If True, the delay between retries is doubled each time               |

LangDB extends the OpenAI-compatible interface and supports most parameters from OpenAI.
